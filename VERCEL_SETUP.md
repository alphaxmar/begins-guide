# วิธีหา Vercel Organization ID และ Project ID

## 1. Organization ID
- ไปที่ Vercel Dashboard
- คลิกที่ Profile/Team ขวาบน
- ไปที่ Settings
- ใน General tab จะเห็น "Organization ID"

## 2. Project ID  
- ไปที่ Project ที่คุณต้องการ deploy
- ไปที่ Settings ของ Project นั้น
- ใน General tab จะเห็น "Project ID"

## 3. หรือสร้าง Project ใหม่ใน Vercel:
- ไปที่ vercel.com/new
- เลือก "Import Git Repository"
- เลือก begins-guide repository
- Deploy ครั้งแรก
- จากนั้นไปเอา Project ID จาก Settings
